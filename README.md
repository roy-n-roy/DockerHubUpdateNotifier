# Docker Hub Update Notifier
Docker hubのリポジトリをタグ単位で監視し、更新があった場合に通知します。
下記のURLから利用できます。

URL: [Docker Hub Update Checker - https://container-image.live-on.net](https://container-image.live-on.net)
